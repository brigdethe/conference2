# Services EJS Standalone

## Run

1. `npm install`
2. `npm start`
3. Open `http://localhost:3000/services`

## Structure

- `app.js` Express entrypoint
- `views/pages/services.ejs` Page composition
- `views/partials/services/*.ejs` Services page partials
- `views/partials/shared/*.ejs` Shared navbar/footer partials
- `external/` and `inline/` static assets from Webflow export
